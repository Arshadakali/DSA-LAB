# DSA LAB

This repository contains coursework and practice code for Data Structures & Algorithms and 8086 assembly exercises. It includes:

- Assembly programs (MASM/TASM and NASM), DOS interrupt demos, number system converters.
- C++ queue implementations and other DSA labs and classwork.
- Helper batch scripts for assembling and running (`run_asm.bat`, `run_nasm.bat`).

## Structure

- `Assembly/` — 8086 assembly sources, MASM/TASM demos, NASM tools.
- `DSA LAB 5/`, `DSALAB Class/`, `DSALAB HOME/` — C++ lab exercises and executables.
- `DSA/` — linked list and queue examples.
- `.vscode/` — editor configuration for C/C++.
